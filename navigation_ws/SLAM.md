# ***SLAM***
## ***一、基本概念***
### 1.1 定义
    SLAM（Simultaneous Localization and Mapping），即同步建图和导航

### 1.2 基础概念
####  1.2.1 地图
+ 尺度地图
+ 拓扑地图
+ 语义地图
+ 栅格地图（占据的概念）

### 1.3 HOW （SLAM流程）
1. 传感器数据 （对外部环境的感知）
2. 前端视觉里程计 （数据对比 获得位置变化）
3. 后端非线性规划
4. 建图
5. 回环检测 （纠正误差）

### 1.4 SLAM算法
#### 1.4.1 基于滤波
扩展卡尔曼滤波 粒子滤波（ROS 中的gmapping、hector_slam）
#### 1.4.2 基于图优化
Cartogapher
